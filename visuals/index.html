Что брать в работу первым

1. Control Tower View

Превращает схему в продукт: KPI, gates, action queue, evidence drawer.

2. Knowledge Graph Hub + NPA Lattice

Фиксирует ядро данных, без которого dashboard будет декоративным.

3. Lifecycle Atlas + Sankey

Дает executive narrative: где цикл, где friction, где осторожно обсуждать эффект.
